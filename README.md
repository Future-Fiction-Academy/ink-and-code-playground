## Welcome 👋

This repository is the public hub for **Future Fiction Academy open-source work**.

Here, we share projects, prototypes, resources, and collaborations created by the FFA community. Everything published here is intended to be publicly available, shared openly, and useful to other creators.

If you're a storyteller, builder, or creative technologist, we'd love for you to join us.

👉 **Join Future Fiction Academy:** [futurefictionacademy.com](https://futurefictionacademy.com/)
